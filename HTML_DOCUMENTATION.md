## Scripts Created

### 1. Browser Export Script (`scripts/browser_export.py`)
- **Purpose:** Transform YAML ingredients to browser-friendly JSON
- **Functionality:**
  - Loads individual YAML files from `data/ingredients/`
  - Extracts searchable fields (name, synonyms, ontology IDs)
  - Aggregates statistics
  - Creates metadata
  - Exports to `docs/data/ingredients.json`
- **Usage:**
  ```bash
  python scripts/browser_export.py
  python scripts/browser_export.py --ingredients-dir data/ingredients --output docs/data/ingredients.json
  ```

### 2. Justfile Commands
- **`just gen-docs`** - Generate LinkML schema documentation
- **`just export-browser`** - Export ingredients to browser JSON
- **`just build-docs`** - Build complete documentation (gen-docs + export-browser)

## Publishing to GitHub Pages

### Quick Setup (3 Steps)

1. **Create GitHub Repository:**
   ```bash
   # Via GitHub web interface or CLI
   gh repo create MediaIngredientMech --public --source=.
   ```

2. **Push Code:**
   ```bash
   git add .
   git commit -m "Add HTML documentation and browser interface"
   git push -u origin main
   ```

3. **Enable GitHub Pages:**
   - Go to repository Settings → Pages
   - Source: Deploy from a branch
   - Branch: `main`
   - Folder: `/docs`
   - Save

**Your site will be live at:**
```
https://YOUR_USERNAME.github.io/MediaIngredientMech/
```

Typically goes live within 1-2 minutes.

### Testing Locally

Before publishing, test locally:

```bash
# Start web server
cd docs
python -m http.server 8000

# Open browser to: http://localhost:8000
```

**Note:** Must use a web server (not `file://`) for JSON loading to work.
